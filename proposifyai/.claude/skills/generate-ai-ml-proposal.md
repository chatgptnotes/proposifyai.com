# Generate AI/ML Project Proposal

Generate comprehensive proposals for AI and machine learning projects including relationship counseling apps, computer vision, NLP, and predictive analytics platforms.

## Context & Purpose

**When to use:** Create technical proposals for AI-powered applications, machine learning platforms, and intelligent automation solutions.

**Target clients:**
- Startups building AI products
- Enterprises adding AI capabilities
- Healthcare/wellness tech companies
- Research institutions
- SaaS companies

**Key project types:**
- AI-powered mobile/web applications
- Computer vision solutions
- NLP and chatbot platforms
- Predictive analytics systems
- Recommendation engines
- Sentiment analysis tools

## Notes for AI Generation

1. **Technical depth varies by audience:**
   - Technical stakeholders: Deep dive into algorithms, architecture
   - Business stakeholders: Focus on ROI, use cases, benefits
   - Mixed audience: Balance technical and business content

2. **AI-specific considerations:**
   - Always mention data requirements
   - Explain model training process
   - Address bias and ethics
   - Discuss ongoing model improvement

3. **Mobile vs Web vs Both:**
   - Adjust deliverables based on platform
   - Mobile: App store deployment, device testing
   - Web: Browser compatibility, responsive design
   - Both: Consistent UX across platforms

4. **Cost factors for AI projects:**
   - Model complexity affects development time
   - Data annotation can be expensive
   - API usage costs (OpenAI, Google AI, etc.)
   - Compute resources for training
   - Ongoing monitoring and retraining

5. **Compliance requirements:**
   - Healthcare: HIPAA compliance
   - Finance: PCI-DSS, SOC 2
   - EU users: GDPR
   - General: Data privacy laws
